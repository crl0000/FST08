- Check git version 
git --version

- Configure git
git config --global user.name "username"
git config user.name
git config --global user.email "email"
git config user.email

- Legends
U - Untracked
A - Added
M - Modified

- Git 3 stage architecture
1) working area
2) Staging area
3) Commit stage

- Check status of the project files and folder
git status

- Add all the files that have been Modified
git add . - all files
git add 'filename.extension'

- Create a commit 
git commit -m "Description."

- Check the history of the changes.
git log (the yellow text is the commit hash)
git log --oneline

Branching and Merging

- List all the available branches
git branch -a

- Create a new branch
git branch [branch-name]

- Switch to a different branch
git switch [branch-name]
git checkout [branch-name]

- Merge a branch
git merge [branch-name]

- Delete a branch
git branch -d [branch-name]
